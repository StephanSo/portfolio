PPE3

Description

Le PPE3 a été réalisé pour aider une organisation (Maison des ligues) à mettre à jour leur application portail web développer en NodeJs, afin de gérer les prestations de reproductions et d'affranchissements.

Méthodes et technologies de travail

Nous étions 4 sur ce projet et nous devions nous partager les tâches afin de pouvoir réaliser le projet dans les temps.
Dans un premier temps nous avons gérer l'authentification des ligues et des conseillers au portail web.
Dans un second temps nous avons gérer les demandes de prestations du cotés des ligues afin que ces dernières puissent demander des prestations au conseiller
La dernière étape consistait à gérer les acceptations des presations du cotés des conseillers et l'affichage des factures pour les ligues. Afin de réaliser ces différente tâches nous avons utilisé différentes méthodes:

  • Nous avons lister les users story afin de valider étapes par étapes toutes les demandes. ( Annexe 1 )
  • Des documentation techniques afin de réutiliser facilement les codes déja développer.
  • Des documentation utilisateur pour que nous puissons valider nos étapes petit à petit.

Contraintes, Difficultés et Solutions

Nous avons rencontrés differentes contraintes:

  • De certaines incompréhension de notre contexte
  • De temps

Pour parrer a ces contraintes on as trouver différente solutions:

  • Essayer de rattrapper le temps perdu sur nos temps libres
  • Parler avec notre demandeurs afin de comprendre les incompréhension
