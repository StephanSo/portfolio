PPE4

Description

Le PPE3 a été réalisé pour aider une organisation (Maison des ligues) à mettre à jour leur application Clémentine développer en Java afin que les Séniors est la possibilité d'être inscrit à des activités sportives. La deuxième partie de l'application celle-ci développer en Android permet au Séniors (Après une connexion) de visualiser les activités auxquels ils sont inscrit.

Méthodes et technologies de travail

Nous étions 4 sur ce projet et nous devions nous partager les tâches afin de pouvoir réaliser le projet dans les temps.
Dans un premier temps nous avons gérer la possibilités au Séniors d'être inscrit à une activitée.
Dans un second temps nous avons gérer l'authentifications des Séniors à l'application android via un webService de l'application NodeJs.
La dernière étape consiste à afficher les activités pour lesquelle un sénior est inscrit. Afin de réaliser ces différente tâches nous avons utilisé différentes méthodes:

  • Nous avons lister les users story afin de valider étapes par étapes toutes les demandes.
  • Des documentation techniques afin de réutiliser facilement les codes déja développer.
  • Des documentation utilisateur pour que nous puissons valider nos étapes petit à petit.

Contraintes, Difficultés et Solutions

Nous avons rencontrés differentes contraintes:

  • De certaines incompréhension de notre contexte
  • De temps

Pour parrer a ces contraintes on as trouver différente solutions:

  • Essayer de rattrapper le temps perdu sur nos temps libres
  • Parler avec notre demandeurs afin de comprendre les incompréhension
