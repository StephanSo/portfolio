Stage 1

Description

Mon projet de stage était de créer une application web afin de paramétrer une application déjà existante du nom de Consélio. Le but était de pouvoir modifier la base de données, d'y ajouter ou de supprimer des valeurs de façon rapide et optimale. Elle devait aussi donner les erreurs inscrites dans la base de données.

Méthodes et technologies de travail

Pour ce projet, j'ai été accompagné d'un expert en .Net. Sur la première semaine de mon stage, il m'a formé aux bases de ce framework.

Durant toute la durée du stage, j'ai du m'auto-former et faire de la veille sur différentes technologies et languages afin de pouvoir réussir le projet:

  • C# (.Net, entityFramework)
  • Html
  • CSS (Bootstrap)
  • JavaScript (Jquery)

Contraintes, Difficultés et Solutions

Durant ce stage j'ai rencontré plusieurs contrainte. La première était le manques de connaissance sur les differents languages que j'ai pus utilisé Ajax, C#, .Net, EntityFramework. Pour ceci je me suis auto-former sur ces differents languages en essayant differentes fonctions

Ensuite j'ai eu des difficultés sur Le language de base de données et de le lier avec mon application. Pour ceci, je me suis aussi auto-former, mais j'ai eu aussi enormement d'aide de mon tuteur de stage.

Le plus gros problème que j'ai pus rencontré était que je n'ai pas utilisé d'outils pour versionner et gérer mon projet
