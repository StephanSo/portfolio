Stage 1

Description

Mon projet de stage était de créé une application web afin de paramétrer une application déjà existante du nom de Consélio. Le but état de pouvoir modifier la base de donnée, d'y ajouter ou de supprimer des valeurs de façon rapide et optimiser. Elle devait aussi donner les erreurs inscrite dans la base de donnée.

Méthode et technologie de travail

Pour ce projet j'ai été accompagner d'un expert en .Net. Sur la première semaine de mon stage, il m'as formé aux bases de ce framework.

Durant toute la durée du stage j'ai du m'auto-former et faire de la veille sur différentes technologie et languages afin de pouvoir reussir le projet:

  • C# (.Net, entityFramework)
  • Html
  • CSS (Bootstrap)
  • JavaScript (Jquery)

Contraintes, Difficultés et Solutions

J'ai pus rencontrer certaines contraintes:

  • De connaissance
  • De compréhension du framework EntityFramework

Pour parrer a ces contraintes je me suis auto-former sur ces connaissances et jai pus obtenir des conseils d'un employé de l'entreprise.

De plus ces contraintes m'ont permis de développer mes connaissances en Javascript (Ajax et Jquery)
